In programming, we use the if..else statement to run a block of code among more than one alternatives. For example, assigning grades (A, B, C) based on the percentage obtained by a student.
  1. the percentage is above 90, assign grade A.
  2. if the percentage is above 75, assign grade B.
  3. if the percentage is above 65, assign grade C.
